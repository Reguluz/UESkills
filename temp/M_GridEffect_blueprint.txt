===============================================================================
材质设置:
- Material Domain: User Interface
- Blend Mode: Additive

节点连接说明（粘贴后自动连接）:
TexCoord → Multiply(×10, GridScale)
Multiply(GridScale) → Floor(整数部分, 网格ID)
Multiply(GridScale) → Frac(小数部分, 网格内坐标)
Frac + Constant(-0.5) → Add(中心偏移)
Floor → MF_Random2D(P) → 随机值

随机值 → If(A) vs Constant(0.6)(B): 主开关
随机值 × RealTime → Multiply(TimeScale) + 随机值 → Add(Phase) → Sin → ×0.75 → ScaleAmp
ScaleAmp × 随机值 → SizeY
Append(ScaleAmp, SizeY) → 基础尺寸

条件分支链: If(随机值 vs 0.8) → If(随机值 vs 0.9) → 最终尺寸变体
If(Main, Output) → Emissive Color
