// ============================================================
// 蓝图复制文本格式核心要点
// ============================================================
//
// 1. 节点连接需要【双层】信息：
//    - Expression= 引用（MaterialExpression 对象层面，用于材质编译）
//    - CustomProperties Pin 的 LinkedTo（EdGraph 层面，用于编辑器图形连线）
//
// 2. LinkedTo 必须是【双向】的：
//    源节点 Output Pin 的 LinkedTo → 目标节点 Input Pin 的 PinId
//    目标节点 Input Pin 的 LinkedTo → 源节点 Output Pin 的 PinId
//
// 3. 每个节点需要：NodeGuid、MaterialExpressionGuid（32位十六进制）
//
// 4. 每个 Pin 需要完整的 CustomProperties Pin 块（PinId、PinName、PinType 等）
//
// 5. 输出 Pin 必须有 Direction="EGPD_Output"，输入 Pin 不需要 Direction 字段
//
// 6. 禁止生成 MaterialExpressionComment（Comment框，会导致UE崩溃）
//
// 7. PersistentGuid 统一使用 00000000000000000000000000000000
